# HH API Lib

Простейшая Python библиотека для получения списка вакансий с hh.ru по официальному API.

## Использование
Необходимо в проекте создать config класса SearchParams со следующими параметрами:
- area_id: str
- vacancy: str
- base_url: str
- access_token: str
- email: str
- period: int (Опционально)
- start_date: str (Опционально)
- end_date: str (Опционально)

Важно создать либо period, либо start_date и end_date в формате "dd.mm.yyyy"

Данный конфиг необходимо передать в качестве аргумента в функцию external_request().
Первым опциональным аргументом является "show_progress=True". Используйте его если у вас есть возможность вывода в консоль и необходимость отслеживать прогресс, с ним будет задействована опциональная библиотека tqdm
Вторым опциональным аргументом является "get_total_found=True". Используйте его если вам необходимо получить кол-во найденных вакансий

Выводом функции является json с полным набором вакансий найденных по запросу "vacancy" 
Опциональным выводом функции является int переменная total_found, обозначающая кол-во найденных вакансий. Кол-во найденных вакансий может отличаться от кол-во полученных вакансий из-за технических ограничений API и содержимого самих вакансий.

Рекомендуется в env разместить ACCESS_TOKEN и email от API hh.ru, в противном случае корректная работа не гарантируется.

!ВАЖНО:
Библиотека на текущий момент не поддерживает сбор более 2000 вакансий по одной профессии за один день
