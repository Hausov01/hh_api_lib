# HH API Lib

Простейшая Python библиотека для получения списка вакансий с hh.ru по официальному API.

## Использование
Необходимо в проекте создать config класса SearchParams со следующими параметрами:
- period: int
- area_id: str
- vacancy: str
- base_url: str
- access_token: str
- email: str

Данный конфиг необходимо передать в качестве атрибута в функцию external_request().
Опциональным аргументом является "show_progress=True". Используйте его если у вас есть возможность вывода в консоль и необходимость отслеживать прогресс

Выходом функции является json с полным набором вакансий найденных по запросу "vacancy"

Рекомендуется в env разместить ACCESS_TOKEN и email от API hh.ru, в противном случае корректная работа не гарантируется
